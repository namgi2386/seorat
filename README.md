## 설치 방법

### Windows

1. zip 파일을 다운로드합니다.
2. 압축을 해제합니다.
3. `pocket seorin.exe`를 실행합니다.

> **"Windows에서 PC를 보호했습니다" 경고가 뜨는 경우**
> 앱이 코드 서명되지 않아 SmartScreen 경고가 표시됩니다.
> 경고 창에서 **"추가 정보"** → **"실행"** 을 클릭하세요.

### macOS

1. `.dmg` 파일을 다운로드합니다.
2. 마운트 후 앱을 Applications 폴더로 이동합니다.
3. 앱을 실행합니다.

> **"손상되었기 때문에 열 수 없습니다" 오류가 뜨는 경우**
> Terminal을 열고 아래 명령어를 실행하세요.
> ```bash
> # Applications 폴더로 이동한 경우
> xattr -cr /Applications/pocket\ seorin.app
>
> # 다운로드 폴더에 있는 경우
> xattr -cr ~/Downloads/pocket\ seorin.app
> ```
> 실행 후 앱을 다시 열면 정상 동작합니다.
