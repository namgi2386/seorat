## 설치 방법

### Windows

1. zip 파일을 다운로드합니다.
2. 압축을 해제합니다.
3. `pocket seorin.exe`를 실행합니다.

> **"Windows에서 PC를 보호했습니다" 경고가 뜨는 경우**
> 앱이 코드 서명되지 않아 SmartScreen 경고가 표시됩니다.
> 경고 창에서 **"추가 정보"** → **"실행"** 을 클릭하세요.

### macOS

1. `.dmg` 파일을 다운로드합니다.
2. 마운트 후 앱을 Applications 폴더로 이동합니다.
3. 앱을 실행합니다.

> **"개발자를 확인할 수 없음" 경고가 뜨는 경우**
> 앱 파일을 **우클릭 → 열기** 선택 후 경고 창에서 **"열기"** 를 클릭하세요.
> 이후에는 일반적으로 실행 가능합니다.
